# GeoAlert 📍

A smart location-based alert Android app that notifies you when you enter or exit a geographic zone.

## Features
- 🗺️ Interactive map with OpenStreetMap
- 🔍 Location search with area boundary visualization
- 📍 Custom geofence zones with adjustable radius
- 🔔 Smart alerts — Notification, Auto-call, or Both
- 📞 Automated call alert when entering/exiting zones
- 📋 Trigger history log with timestamps
- 🔄 Persistent geofences that survive device reboots
- 📵 Works offline (map tiles cached)

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- Android Geofencing API
- OSMDroid (OpenStreetMap)
- Retrofit + Nominatim API
- Coroutines + StateFlow

## Setup
1. Clone the repo
2. Open in Android Studio
3. Run on a physical Android device (API 26+)

## Note
Geofencing requires a real Android device — emulators do not support it fully.
